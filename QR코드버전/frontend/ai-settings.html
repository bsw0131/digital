API 키 받는 방법

  1. OpenAI 플랫폼에 로그인합니다.
  2. API Keys 메뉴에서 새 Secret key를 만듭니다.
  3. 생성된 키를 복사해 이 화면의 OpenAI API 키 칸에 붙여넣습니다.
  4. AI 설정 저장을 누르면 이후 QR로 접속한 학생들도 온라인 AI 추천을 사용합니다.
API 키 만들기 사용량 확인

API 기능과 주의사항

  • API 키는 이 프로그램이 OpenAI 모델을 호출하기 위한 비밀 비밀번호입니다.
  • 학생 화면에는 API 키가 보이지 않고, 이 PC의 data/settings.json에만 저장됩니다.
  • 온라인 AI는 탐구주제 추천부터 보고서 초안까지 더 구체적인 결과를 만들 때 사용됩니다.
  • 인터넷이 안 되거나 키가 잘못되면 자동으로 오프라인 추천 엔진을 사용합니다.
  • API 사용량에 따라 OpenAI 계정에 비용이 발생할 수 있으니 사용량 페이지를 확인하세요.
