# AI 탐구메이트 사용설명서

## 학생 사용 흐름

1. `학생으로 시작`을 누릅니다.
2. 이름과 학번을 입력합니다.
3. 관심 태그를 선택하고 세부 관심사를 입력합니다.
4. 추천된 탐구 주제 중 하나를 선택합니다.
5. 탐구 마법사에서 계획서, 자료조사, 설문/인터뷰, 탐구일지, 보고서 초안을 작성합니다.

## 교사 사용 흐름

1. `교사 대시보드`를 누릅니다.
2. 비밀번호 `teacher1234`로 로그인합니다.
3. 학생별 주제, 진행률, 적합도를 확인합니다.
4. 필요한 학생에게 피드백을 남깁니다.

## 실행 주소

서버 실행 후 브라우저에서 `http://127.0.0.1:8000`으로 접속합니다.
