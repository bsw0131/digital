# AI 탐구메이트

중학생의 관심사에서 출발해 탐구 주제 추천, 탐구계획서 작성, 자료조사, 설문/인터뷰, 탐구일지, 보고서 초안 생성을 돕는 웹 프로그램입니다.

## 실행 방법

Windows에서 `run_server.bat`를 실행하면 필요한 패키지를 설치한 뒤 FastAPI 서버가 실행됩니다.

브라우저에서 다음 주소로 접속합니다.

```text
http://127.0.0.1:8000
```

## 폴더 구조

```text
backend/   FastAPI 서버, DB, 추천 엔진
frontend/  학생/교사 화면과 브라우저 코드
docs/      사용 설명 문서
deploy/    실행 안내
```

## 교사 로그인

기본 비밀번호는 `teacher1234`입니다.

## OpenAI API 사용

`backend/.env` 파일에 `OPENAI_API_KEY`를 넣으면 온라인 AI 추천을 사용할 수 있습니다. 키가 없으면 오프라인 추천 엔진으로 동작합니다.
