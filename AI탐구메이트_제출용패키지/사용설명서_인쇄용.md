# AI탐구메이트 사용설명서

## 1. 실행하기

`AI탐구메이트.exe`를 더블클릭합니다. 브라우저가 자동으로 열리며, 열리지 않을 경우 `http://127.0.0.1:8000`으로 접속합니다.

## 2. 학생 사용 방법

1. 메인 화면에서 `학생으로 시작`을 누릅니다.
2. 학번과 이름을 입력합니다.
3. 관심 태그를 선택하거나 직접 관심사를 입력합니다.
4. 직접 입력은 쉼표로 구분할 수 있으며, 선택 관심사와 직접 입력 관심사를 합쳐 최대 2개까지 융합할 수 있습니다.
5. 추천된 탐구 주제 중 하나를 선택합니다.
6. 탐구계획서, 자료조사, 설문/인터뷰, 탐구일지, 보고서 초안을 작성합니다.
7. 기존 탐구는 이어서 진행하거나 삭제할 수 있습니다.

## 3. 교사 사용 방법

1. 메인 화면에서 `교사 대시보드`를 누릅니다.
2. 비밀번호 `teacher1234`로 로그인합니다.
3. 학생별 주제, 진행률, 최근 진행 상황을 확인합니다.
4. 저장 이력은 펼쳐서 전체 최근 기록을 확인할 수 있습니다.
5. 필요한 경우 교사 피드백을 입력하고 저장합니다.
6. `PDF로 저장` 버튼으로 학생 작업 상황을 PDF로 저장합니다.
7. 새 수업 또는 새 대회 준비 시 `학생자료 초기화` 버튼으로 데이터를 초기화할 수 있습니다.

## 4. AI 설정

1. 교사 대시보드 옆의 `AI 설정` 버튼을 누릅니다.
2. 교사 비밀번호를 입력해 설정 화면에 들어갑니다.
3. 온라인 AI 사용 여부, OpenAI API 키, 모델명을 저장합니다.
4. API 키가 없거나 인터넷 연결이 불안정하면 오프라인 추천 엔진이 자동으로 사용됩니다.

## 5. 데이터 저장 위치

학생 자료와 교사 피드백은 `data/inquiry_mate.db`에 저장됩니다.

AI 설정은 `data/settings.json`에 저장됩니다.

제출 전에 필요한 데이터가 들어 있는지 확인하세요.

## 6. 연구대회 시연 안내

본 프로그램은 exe 실행형으로 제출 및 시연할 수 있습니다. 내부적으로는 로컬 웹앱 서버를 실행하고 브라우저에서 학생 탐구실과 교사 대시보드를 사용하는 구조입니다.

수업 적용 시에는 같은 네트워크 공유, ngrok 터널링, 또는 실제 웹 배포 방식으로 학생 접속형 수업으로 확장할 수 있습니다.
