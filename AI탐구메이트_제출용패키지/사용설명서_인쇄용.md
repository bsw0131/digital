# AI탐구메이트 사용설명서

## 1. 실행하기

`AI탐구메이트.exe`를 더블클릭합니다. 브라우저가 자동으로 열리며, 열리지 않을 경우 `http://127.0.0.1:8000`으로 접속합니다.

## 2. 학생 사용 방법

1. 메인 화면에서 `학생으로 시작`을 누릅니다.
2. 이름과 학번을 입력합니다.
3. 관심 태그를 선택하고 세부 관심사를 입력합니다.
4. 추천된 탐구 주제 중 하나를 선택합니다.
5. 탐구계획서, 자료조사, 설문/인터뷰, 탐구일지, 보고서 초안을 작성합니다.
6. 기존 탐구 카드의 `진행 상황` 칸에 현재 작업 내용을 적고 저장합니다.

## 3. 교사 사용 방법

1. 메인 화면에서 `교사 대시보드`를 누릅니다.
2. 비밀번호 `teacher1234`로 로그인합니다.
3. 학생별 주제, 진행률, 진행 상황을 확인합니다.
4. 필요한 경우 피드백을 입력하고 저장합니다.
5. `PDF로 저장` 버튼으로 학생 작업 상황을 PDF로 저장합니다.
6. 새 수업 또는 새 대회 준비 시 `학생자료 초기화` 버튼으로 데이터를 초기화할 수 있습니다.

## 4. 인터넷 연결 상태

메인 화면에는 인터넷 연결 여부가 표시됩니다.

- 초록색 동그라미: 인터넷이 연결되어 있음
- 빨간색 동그라미: 인터넷이 연결되어 있지 않음

인터넷이 없어도 기본 오프라인 추천 기능은 사용할 수 있습니다.

## 5. 데이터 저장 위치

학생 자료와 교사 피드백은 `data/inquiry_mate.db`에 저장됩니다. 제출 전에 필요한 데이터가 들어 있는지 확인하세요.
