AI탐구메이트 실행 방법

1. AI탐구메이트.exe를 더블클릭합니다.
2. 잠시 기다리면 기본 브라우저가 자동으로 열립니다.
3. 브라우저가 열리지 않으면 주소창에 아래 주소를 직접 입력합니다.

   http://127.0.0.1:8000

4. 학생은 '학생으로 시작'을 누르고 이름과 학번을 입력합니다.
5. 교사는 '교사 대시보드'를 누르고 비밀번호 teacher1234로 로그인합니다.

주의 사항

- AI탐구메이트.exe와 data 폴더는 같은 폴더 안에 있어야 합니다.
- 학생 자료는 data/inquiry_mate.db에 저장됩니다.
- USB에서 실행할 수 있지만, 일부 학교 PC 보안 정책에 따라 exe 실행이 차단될 수 있습니다.
- 인터넷이 없으면 오프라인 추천 엔진으로 동작합니다.
- OpenAI API 키가 없으면 온라인 AI 추천 대신 오프라인 추천이 사용됩니다.
