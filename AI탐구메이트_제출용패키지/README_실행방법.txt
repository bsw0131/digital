AI탐구메이트 실행 방법

1. AI탐구메이트.exe를 더블클릭합니다.
2. 잠시 기다리면 기본 브라우저가 자동으로 열립니다.
3. 브라우저가 열리지 않으면 주소창에 아래 주소를 직접 입력합니다.

   http://127.0.0.1:8000

4. 학생은 '학생으로 시작'을 누르고 학번, 이름 순서로 입력합니다.
5. 교사는 '교사 대시보드'를 누르고 비밀번호 teacher1234로 로그인합니다.
6. 온라인 AI를 쓰려면 교사 대시보드의 'AI 설정'에서 API 키와 모델을 저장합니다.

이번 제출용 버전에 반영된 주요 기능

- 교사 대시보드에서 루브릭 평가는 제외하고 교사 피드백 입력만 남겼습니다.
- 학생 진행 상황은 최근 저장 1개를 먼저 보여주고, 펼치면 저장 이력을 확인할 수 있습니다.
- 학생 탐구실 로그인 입력 순서를 학번, 이름 순서로 정리했습니다.
- 관심사와 직접 입력 관심사를 합쳐 최대 2개까지 융합해 탐구 주제를 추천합니다.
- 직접 입력 3개 등 조건에 맞지 않는 경우 오류 메시지를 보여줍니다.
- 자료조사 가이드, 설문 문항, 인터뷰 질문을 주제에 맞게 더 구체적으로 생성합니다.
- 학생의 기존 탐구 카드에 삭제 기능을 추가했습니다.
- 교사 AI 설정을 별도 화면으로 분리했습니다.

주의 사항

- AI탐구메이트.exe와 data 폴더는 같은 폴더 안에 있어야 합니다.
- 학생 자료는 data/inquiry_mate.db에 저장됩니다.
- 온라인 AI 설정은 data/settings.json에 저장됩니다.
- 인터넷이 없거나 API 키가 없으면 오프라인 추천 엔진으로 동작합니다.
- USB에서 실행할 수 있지만, 일부 학교 PC 보안 정책에 따라 exe 실행이 차단될 수 있습니다.
- exe 창을 닫으면 프로그램 서버도 함께 종료됩니다.

수업 확장 안내

- 같은 와이파이에 있는 학생에게 접속시키려면 교사 PC에서 서버를 0.0.0.0으로 열어야 합니다.
- 다른 지역 학생에게 접속시키려면 ngrok 또는 실제 웹 배포가 필요합니다.
- 연구대회 제출/시연은 exe 실행형, 실제 수업 운영은 웹 배포형으로 확장할 수 있습니다.
